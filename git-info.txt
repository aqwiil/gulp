git init - созадть репозіторій(заставити гіт слідкувати за файлами)
git config --global user.name "aqwiil" - задать локально імя репозіторія(проекта)
git config --local user.email "aqwiil1337@gmail.com" - задать локально емайл репозіторія(проекта)
local/global - локально/глобально

git status - дізнатись статус(чи слідить гіт за файлами
   чи були добавленні якісь коміти, або контрольні точки)

git add -A(можливо замість -А вставити назву окремого файла) - заставили гіт слідкувати за всіма файлами для можливості їх
   закомітити

git commit -a(дя всі файлів і опять же можна для окремого) 
   -m"first commit"(-m"" назва коміта)

git log - подивитись які коміти були создані
git pull - вигрузити всі коміти з одно компа на інший(коміти які добавились після пред ідущой вигрузки)
